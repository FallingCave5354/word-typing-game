# C++ Word Typing Game
A console-based application designed to help users improve their typing capabilities, speed, and accuracy. The game challenges players to type randomly generated words from a dictionary of nearly 3,000 words.

# Game Features

   - Massive Dictionary: Includes a library of 2,964 words ranging from simple common words to complex terms.
   - 10 Difficulty Modes: Ranges from "Beginner" to "Insane," plus a "Custom Mode" for specific training sessions.
   - Real-time Feedback: Notifies you immediately if your input does not match the target word.
   - Post-Game Statistics: displays your final score, total correct/wrong words, and a precise accuracy percentage calculated to 4 significant digits.
   - Replay System: Quickly restart the game without reopening the application.



# How to Play
1. Launch the Game: Run the compiled executable file.
2. Select Difficulty: You will be prompted to select a difficulty level between 1 and 10.
3. Type the number corresponding to your desired mode and press Enter.
4. Type the Words: A random word will appear on the screen (e.g., 1. Type "abandon").
5. Type the word exactly as shown next to Answer: and press Enter.
6. Note: In Custom Mode, you will be asked to manually input the "Amount of Words" you wish to play.

7. Review Results:
   - If correct, the game moves to the next word.
   - If wrong, the game displays "Your answer didn't match the word!" before moving on.
   - View Score: Once the session is finished, review your accuracy and score.

8. Play Again: Type Y (or y) to restart the game, or any other key to exit.
